# Laravel 10 Task Manager

This repository is a Laravel 10 task manager application.

## Setup

1. Install PHP dependencies:
   ```bash
   composer install
   ```
2. Install Node dependencies and build assets:
   ```bash
   npm install
   npm run build
   ```
3. Run database migrations:
   ```bash
   php artisan migrate
   ```
4. Run the test suite:
   ```bash
   php artisan test
   ```
